=head1 NAME

Slack - A slacked web application framework based on Plack

=head1 VERSION

This document describes Slack version 0.02

=head1 SYNOPSIS

  $ cat app.psgi
  package MyApp;
  use Slack qw(App);
  __PACKAGE__->new->to_app;

  $ cat MyApp/Root.pm
  package MyApp::Root;
  use Slack qw(Controller);

  action hello => sub {
      res->body('Hello, world!');
  };

  1;

  $ plackup
  HTTP::Server::PSGI: Accepting connections at http://0:5000/

  $ curl http://127.0.0.1:5000/hello
  Hello, world!
